# Agentic RAG-VLM

**Agentic RAG-VLM: Affordance-Aware Retrieval-Augmented Generation with Self-Reflective Planning for Robotic Grasping**

> Submitted 

---

## Overview

This repository contains the code and experimental framework for **Agentic RAG-VLM**, a unified framework for robotic grasping that bridges semantic understanding and physical manipulation by integrating **Retrieval-Augmented Generation (RAG)** with **Vision-Language Models (VLMs)** and **self-reflective agentic planning**.

<p align="center">
  <img src="results/iros2026/figures/system_overview.png" alt="System Overview" width="800"/>
</p>

### Key Components

1. **Hierarchical Affordance-Aware RAG (HAA-RAG):** Encodes object affordance descriptors—including affordance type, material, fragility, and graspable region—and retrieves grasp strategies by functional affordance compatibility rather than visual appearance.
2. **Scene Graph Constraint Reasoner:** Constructs spatial relationship graphs from VLM perception and translates manipulation constraints into concrete grasp parameter adjustments.
3. **Agentic Self-Reflective Pipeline:** Structured failure diagnosis with multi-level adaptive retry for closed-loop grasp refinement.

## Installation

### Prerequisites

- Python >= 3.10
- CUDA >= 12.0
- PyTorch >= 2.0

### Setup

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/Agentic-RAG-VLM.git
cd Agentic-RAG-VLM

# Install dependencies
pip install -r robot_grasp_rag/requirements.txt

# Download model weights (not included in repo due to size)
# Qwen3-VL-8B: https://huggingface.co/Qwen/Qwen3-VL-8B
# Place under models/Qwen3-VL-8B/
```

### Configuration

Edit `robot_grasp_rag/config/config.yaml` to set your model path:

```yaml
vlm:
  model_path: "/path/to/your/Qwen3-VL-8B"
```

## Experimental Results

Our method is evaluated on a multi-task benchmark spanning three difficulty levels:

| Task Type                       | Description                         | Ours            |
| ------------------------------- | ----------------------------------- | --------------- |
| **Single Grasp (SG)**     | Pick one target from clutter        | **91.7%** |
| **Interactive Task (IT)** | Multi-step with spatial constraints | **64.2%** |
| **Long-Horizon (LH)**     | Sequential table clearing           | **66.7%** |
| **Overall**               | Weighted average                    | **78.3%** |

Detailed results including ablation studies, OOD generalization, and latency analysis are available in `results/iros2026/`.

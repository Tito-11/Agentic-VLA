## 📋 技术方案概述

本项目实现了 **Unified Agentic RAG-VLM**，一个基于检索增强的机器人抓取智能规划系统，整合了四个核心创新：

### 🎯 核心创新点

| 创新 | 描述 | 效果 |
|------|------|------|
| **HAA-RAG** | 层次化可供性感知检索 (类别→可供性→视觉) | P@1 +19.4% |
| **Multi-Modal RAG** | 多模态联合检索 (Text+Vision+Geometry) | P@1 +78.4% |
| **Scene Graph RAG** | 场景图约束推理 | 碰撞避免 +78.8% |
| **Agentic Pipeline** | ReAct + Self-Reflection + Adaptive Retry | 成功率 +81% |

### 📊 关键实验结果

| 指标 | 数值 | vs Baseline |
|------|------|-------------|
| **SOTA 对比** | 83.3% | +81% vs CLIPort |
| **消融实验** | 82.2% ± 1.9% | +101% vs NoRAG (p<0.001) |
| **OOD 泛化** | 66.7% Far-OOD | 退化率仅 6.7% |
| **VLM 加速** | 155.5 t/s | 1.78x |
| **显存占用** | 6.7 GB | -60% |

## 📚 论文创新点总结

1. **RAG for Robotic Grasping**: 首次将 RAG 技术系统性应用于机器人抓取任务
2. **Dual-Path Retrieval**: 语义+视觉双路检索，解决视觉相似但语义不同的物体误匹配问题
3. **Experience-Driven Few-Shot**: 基于历史成功经验的 In-Context Learning
4. **Edge-Efficient VLM**: INT4 + FP8 KV Cache 实现端侧实时推理
